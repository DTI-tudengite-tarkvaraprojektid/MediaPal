What is MediaPal?

MediaPal is a plug-in that allows any Excel or Google Sheets data to be interactively portrayed on a map. For the main example, we have used a Google Sheets database, thats been imported as .csv to our plugin.

Due to the current overwhelmingly diverse streaming market, we have created a map in which any user can search for their specified country, streaming category or even movie genre and find the informaton necessary. And all this information directly from a simple Google Sheets file!

Features

Create Maps!

Create beautiful and customizable map interactive interfaces via Mapbox

Store the location!

Pinpoint an exact location on a map and store the value from your database to portray whatever you need in any way you need

Dont worry about security!

The plug-in will only have to connect to a FireBase database via a simple .html file in your desktop folder


The MediaPal plug-in consists of both Front-End and Back-End sections. In Front-end side, there is a Platform Manager which allows the user to load any Excel or Sheets file to the server. In specific, the file will be reformatted as JSON and then sent to the server.

Secondly, the plugin allows the developer to integrate an extra website to their own page via the Mapbox API and thus search the data loaded by Platform Manager

In the Back-End side, there needs to be done a Firebase Database Setup where the JSON formatted file is sent and received by Firebase Functions Endpoints, which creates the functionality necessary.

How to setup the MediaPal plugin for your own data

In order to setup the MediaPal plugin, you will first need to download the folder which contains all necessary files here.

Then you'll need to setup a Firebase project and a Firebase Database and finally Firebase Functions. Once that is done, create a new database collection called "database" and create an empty file called "EXMtb47HNp7j3hTdhhOF". Once you've done that, you can load Firebase functions from your index.js file by can replacing the index.js Firebase has given you.

Now that all the Firebase stuff is created, you can add the plugin to your created website. Create a Mapbox account. Use the plugin folders style.css and mediapal.js files in your .html file.


Lastly, to start the plugin, you will need to setup this piece of code 

mediapal.open(document.getElementById("created-element-ID"), 
{
    URL: "serveri GET endpoint URL",
    MAPBOX_TOKEN: "MAPBOX Token",
    MAPBOX_SETTINGS: {
        container:'map',
        style: 'Mapbox Style Token',
        center: [-100.04, 38.907],
        zoom: 3
    },
    COUNTRY_COLOR: 'rgba(23, 236, 105, 0.4)',
    COUNTRY_OUTLINE_COLOR: 'rgba(0, 4, 2, 1)'
});

You will need to change all of the pink highlighted attributes

  • URL - the Servers endpoint URL where the .csv file is acquired in JSON format
  • MAPBOX_TOKEN - Mapbox token from Account Settings
  • MAPBOX_SETTINGS - Mapbox map initialisation settings from here
  • COUNTRY_COLOR - Country colors in RGBA format, that you would like to use
  • COUNTRY_OUTLINE_COLOR - Country outline colors in RGBA format, that you would like to use

Then you're all set up! Have fun!
